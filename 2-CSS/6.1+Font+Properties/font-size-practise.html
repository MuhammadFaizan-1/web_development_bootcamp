Types of Font sizes

  1. Px
    • This is an absolute size.
    • This is 1/92th part of an inch.
  2. Pt
    • This is an absolute size.
    • This is 1/72th part of an inch.
    • This is used in MS WORD.
  3. em
    • This is a relative size.
    • This is relative to the parent element size.
    • 1em = 100 % of parent size, 2 em = 2 * size of parent element
  4. rem
    • This is a relative size.
    • This is relative to the root element size, mainly HTML tag.
    • 1rem = 100 % of root size, 2 rem = 2 * size of root.
    • This is most suitable type to use.
  5. Fixed
    • This is an absolute size.
    • This has fixed sizes such as large, x-large,xx-large
